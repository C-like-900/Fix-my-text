** "Fix my text" v4 **

Программа "Fix my text" (Далее "FMT") предназначена для проведения операций с выделенным текстом.
//Программа FMT пользуется буфером обмена, но сохраняет исходные текстовые данные.

Для работы с "FMT" необходимо:
 - Выделить текст
 - Использовать одно из сочетаний клавиш (сочетания описанны ниже)

Возможности "FMT" (в данной версии):
 - Замена языка у выделенного текста (Win+Ctrl)  //Помощь при неправильной раскладке
 - Инверсия регистра букв (Win+Alt)  //Помощь при неправильном Caps Lock

Удачного пользования :)
