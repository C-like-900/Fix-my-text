** Word++ v2 **

Программа Word++ предназначена для проведения операций с выделенным текстом.
//Программа Word++ пользуется буфером обмена, но сохраняет исходные текстовые данные.

Для работы необходимо:
 - Выделить текст
 - Зажать один из нескольких сочетаний клавиш (сочетания описанны ниже)

Возможности Word++ (в данной версии):
 - Замена языка у выделенного текста (Win+F4)  //Помощь при неправильной раскладке
 - Очитска по грамматике (уборка лишних пробелов/добавление пробела перед знаками препинания) (Win+F11)
 - Инверсия регистра букв (Win+F12)  // Помощь при неправильном Caps Lock

Программа запускается в фоновом режиме, поэтому чтобы закрыть нужно ЗАЖАТЬ кнопку Esc (1.5 сек).
