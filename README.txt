
    FIX/FIND MY TEXT - FMT

Данная программа, включает в себя набор функций для разнообразной работы с выделенным текстом.
Она позволяет изменять языковой формат, регистр символов, отправлять текстовые запросы в поисковые системы и переводить текст через браузер.
__________________________________________________________________________________________________________________________________________________________________________

Подробнее про функции приложения:
1.	Смена языковой раскладки [Win + Shift + Z].
Описание: Функция меняет пословно языковую раскладку. Для этого проверяется каких больше символов (русских или английских) и после символы меняются на противоположные.
Пример: Ghbdtn vbh! -> Привет мир!

2.	Смена буквенного регистра [Win + Shift + X].
Описание: Функция посимвольно меняет буквенный регистр путем математической проверки.
Пример: сЛУЧЙНЫЙ НАБОР СЛОВ -> Случйный набор слов

3.	Поиск текста в браузере [Alt + Win + Z].
Описание: Функция отправляет интернет запрос в поисковик с выбранным текстом.
Пример: «https://ya.ru?q=» + текст

4.	Открытие текста через переводчик [Alt + Win + X].
Описание: Функция отправляет интернет запрос в поисковик с выбранным текстом.
Пример: «https://translate.google.ru/?sl=auto&tl=ru&text=» + текст
