** Word++ v2 **

Программа Word++ предназначена для проведения операций с выделенным текстом.
//Программа Word++ пользуется буфером обмена, но сохраняет исходные текстовые данные.

Для работы необходимо:
 - Выделить текст
 - Зажать один из нескольких сочетаний клавиш (сочетания описанны ниже)
 - Отпустить все клавишы

Возможности Word++ (в данной версии):
 - Замена языка у выделенного текста (Win+Ctrl)  //Помощь при неправильной раскладке
 - Инверсия регистра букв (Win+Alt)  //Помощь при неправильном Caps Lock
 - Очитска по грамматике (уборка лишних пробелов/добавление пробела перед знаками препинания) (Win+Esc)

Удачного пользования :)
